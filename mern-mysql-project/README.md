# MERN Stack Authentication with MySQL CRUD Application

## Project Description
This project is a full stack web application developed using React.js, Node.js, Express.js, and MySQL database.

The application allows users to:
- Register a new account
- Login securely using JWT authentication
- Add items
- View items
- Update items
- Delete items
- Search items
- Export items to CSV
- Toggle dark mode
- Logout securely

Passwords are encrypted using bcryptjs and authentication is handled using JSON Web Tokens (JWT).
